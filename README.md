# NumPy Student Marks Analysis

This project analyzes student marks using the NumPy library.

## Project Overview
A 1D NumPy array representing student marks was analyzed to understand class performance.

## Analysis Performed
- Minimum marks
- Maximum marks
- Average marks
- Pass and fail student identification
- Class passing rate
- Class failure rate

## Tools Used
- Python
- NumPy
- Jupyter Notebook
